📄 Extracción Inteligente de Datos de Facturas Eléctricas

Proyecto de machine learning aplicado a OCR + NLP, centrado en extraer automáticamente información clave de facturas eléctricas en PDF (como fechas, consumo, CUPS o importes), usando un modelo T5 ajustado y OCR especializado con Doctr.

⚙️ Tecnologías Utilizadas

Python T5 (Transformers) Doctr OCR Streamlit PyMuPDF HuggingFace Datasets

🧠 Flujo del Proyecto

  1. Generación de Dataset: se procesan los PDFs con Doctr y se estructuran los datos OCR en JSONL para etiquetado.
  2. Etiquetado manual: se visualizan y etiquetan fácilmente las facturas con Streamlit, y el progreso se guarda automáticamente.
  3. Verificación: comprobador interactivo del dataset etiquetado para revisión y control de calidad.
  4. Entrenamiento: fine-tuning de un modelo T5-base con los textos OCR y las etiquetas.
  5. Inferencia: extracción automática de campos clave desde facturas nuevas, usando lógica híbrida: T5 + validaciones OCR específicas.

📤 Campos Extraídos

  • fechaDesde / fechaHasta
  • cups
  • consumo (kWh)
  • euroConsumo (€)
  • total (€)
  • excesoPotencia (€)

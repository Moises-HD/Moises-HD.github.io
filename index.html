📄 Extracción inteligente de datos de facturas eléctricas

Python · T5 · Transformers · Doctr · OCR · PyMuPDF

Proyecto que utiliza un modelo T5 fine-tuned junto a OCR (Doctr) para analizar facturas eléctricas en PDF y extraer campos clave como CUPS, consumo, importe total y excesos de potencia. Incluye lógica OCR personalizada, validación numérica y reglas específicas para casos como facturas rectificativas o sin cargos.

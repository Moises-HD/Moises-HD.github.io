## 📁 Proyectos

### 🔍 [Factura ML OCR](./Proyectos/Factura-ML-OCR)

Extracción automática de datos estructurados desde facturas eléctricas en PDF. Combina modelos OCR con un modelo T5 entrenado específicamente para identificar:

- Periodo de facturación
- CUPS
- Consumo energético
- Importe por energía activa
- Total facturado
- Excesos de potencia

Incluye etiquetado visual con Streamlit, entrenamiento y modelo inferencial.

📸 Capturas de precisión en [`/Proyectos/Factura-ML-OCR/capturas/`](https://github.com/Moises-HD/Moises-HD.github.io/tree/main/Proyectos/Factura-ML-OCR/capturas)
